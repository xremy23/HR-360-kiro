NOTIFICATION CHANNELS (TO CONFIGURE)
================================================================================

To add notifications to alerts, configure channels:

EMAIL:
  gcloud alpha monitoring channels create \
    --display-name="Email Alert" \
    --type=email \
    --channel-labels=email_address=your-email@example.com

SLACK:
  gcloud alpha monitoring channels create \
    --display-name="Slack Alert" \
    --type=slack \
    --channel-labels=channel_name=#alerts

PAGERDUTY:
  gcloud alpha monitoring channels create \
    --display-name="PagerDuty Alert" \
    --type=pagerduty \
    --channel-labels=service_key=YOUR_SERVICE_KEY

Then update alert policies to use these channels.
