API ENDPOINT REFERENCE
================================================================================

CREATE ORGANIZATION
Request:  POST /org
Auth:     Bearer {JWT_TOKEN}
Body:     { name: "Company Name", emailDomain?: "domain.com", logo?: "url" }
Response: { id, name, emailDomain, inviteCode, logo, createdAt }
Status:   201 Created

GET ORGANIZATION
Request:  GET /org
Auth:     Bearer {JWT_TOKEN}
Response: { id, name, emailDomain, inviteCode, logo, createdAt }
Status:   200 OK

UPDATE ORGANIZATION
Request:  PUT /org
Auth:     Bearer {JWT_TOKEN} (Admin role required)
Body:     { name?: "New Name", logo?: "url" }
Response: { id, name, emailDomain, inviteCode, logo, createdAt }
Status:   200 OK

INVITE USER
Request:  POST /org/invite
Auth:     Bearer {JWT_TOKEN} (Admin role required)
Body:     { email: "user@example.com", role: "employee" }
Response: { email, inviteCode, organization }
Status:   200 OK
Note:     Email sent to user with invitation link

LIST ORGANIZATION USERS
Request:  GET /org/users?teamId=X&role=admin&limit=50&offset=0
Auth:     Bearer {JWT_TOKEN} (Admin role required)
Response: { data: [{id, email, firstName, lastName, role, teamId}], total, page }
Status:   200 OK

REMOVE USER FROM ORGANIZATION
Request:  DELETE /org/users/{userId}
Auth:     Bearer {JWT_TOKEN} (Admin role required)
Response: {}
Status:   200 OK
Note:     Cannot remove yourself

SWITCH ORGANIZATION
Request:  POST /org/switch
Auth:     Bearer {JWT_TOKEN}
Body:     { orgId: "org-id" }
Response: { id, name, emailDomain, inviteCode, logo, createdAt }
Status:   200 OK
Note:     For future multi-org support

LIST ORGANIZATION TEAMS
Request:  GET /org/teams
Auth:     Bearer {JWT_TOKEN}
Response: [{ id, memberCount }]
Status:   200 OK

TROUBLESHOOTING GUIDE
================================================================================

ISSUE: Organization Settings button not visible
CAUSE: User is not admin/hr
FIX: Verify user role in JWT token or database

ISSUE: Cannot create organization (form submitted but nothing happens)
CAUSE: Backend error or network issue
FIX: Check browser console for errors, verify backend is running

ISSUE: Invitation email not received
CAUSE: Email service configuration issue
FIX: 
  1. Check backend/.env EMAIL_USER and EMAIL_PASSWORD
  2. Verify Gmail app password is correct
  3. Check Gmail spam folder
  4. Verify recipient email is correct

ISSUE: Members list not loading
CAUSE: Database connection or API error
FIX: Check backend logs, verify user has admin role

ISSUE: Cannot remove user
CAUSE: User is trying to remove themselves or lacks permissions
FIX: Verify user doing removal has admin role, different user being removed

ISSUE: White screen when visiting organization settings
CAUSE: Frontend error or not authenticated
FIX:
  1. Check browser console for JavaScript errors
  2. Verify user is logged in
  3. Clear browser cache and reload
  4. Try different browser
