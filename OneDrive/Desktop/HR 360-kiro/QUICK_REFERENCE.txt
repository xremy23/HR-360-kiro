🏢 ORGANIZATION MANAGEMENT - USER WALKTHROUGH
================================================================================

CREATE ORGANIZATION:
  1. Login → Settings → 🏢 Organization Settings
  2. Click "Create Organization"
  3. Enter name (auto-fills domain)
  4. Submit → You're now admin!
  5. Share invite code with team

INVITE TEAM MEMBER:
  1. Organization Settings → Invite tab
  2. Enter email, select role
  3. Click "Send Invitation"
  4. They receive magic link email
  5. They click link to join

MANAGE MEMBERS:
  1. Organization Settings → Members tab
  2. View all members
  3. Click "Remove" to remove user

⚡ API ENDPOINTS
================================================================================
POST /org                    → Create organization
GET /org                     → Get organization
PUT /org                     → Update organization
POST /org/invite             → Send invitation email
GET /org/users               → List members (admin only)
DELETE /org/users/{userId}   → Remove member (admin only)
POST /org/switch             → Switch organization
GET /org/teams               → List teams
