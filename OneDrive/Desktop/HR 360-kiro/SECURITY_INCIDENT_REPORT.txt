LESSONS LEARNED
================================================================================

What Went Wrong:

  2. No pre-commit hooks to prevent secret commits
  3. No automated secret scanning in CI/CD
  4. Credentials stored in plain text in .env files

What To Do Better:

  2. Use git-secrets or TruffleHop to scan for secrets
  3. Use environment variables instead of .env files in production
  4. Use Cloud Secret Manager for sensitive data
  5. Enable branch protection and require code review
  6. Implement pre-commit hooks
  7. Regular security audits of repository
